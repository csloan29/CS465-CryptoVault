# Crypto vault: all algorithm projects for BYU CS 465

### Requirements to Run:
- Python 3
- Python Modules by Project:
    AES:            (numpy)
    HashAttack:     (numpy, matplotlib, random, hashlib, binascii, bitarray)
    Mac Attack:     (binascii, struct, hashlib)
    DiffieHellman:  (math)
    RSA:            (math, pycryptodome)

### To Run:
`python3 RSA.py`
